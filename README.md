# Clean White Theme for Hugo Extend

基于 [hugo-theme-cleanwhite](https://github.com/zhaohuabing/hugo-theme-cleanwhite) 扩展了 `Latex` 语法。

行内公式语法用单 `$` 括起来:

```bash
$\hat{y} =x\  \times w\  +\  b$
```

`$\hat{y} =x\  \times w\  +\  b$`

行间公式语法用双 `$` 括起来:

```bash
$$\hat{y} =x\  \times w\  +\  b$$
```

`$$\hat{y} =x\  \times w\  +\  b$$`

> MacOS 用户推荐 `App Store` 的 `XFormula` 编写并输出 `Latex`
